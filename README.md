# Tabba Marketing Website

This folder contains the standalone marketing website for the Tabba Chrome extension.

## Structure

- `index.html` - Main landing page
- `styles/` - CSS files matching the extension's design system
  - `variables.css` - Design tokens (colors, spacing, typography, etc.)
  - `base.css` - Base styles and reset
  - `landing.css` - Landing page specific styles

## Design System

The website uses the same design system as the extension:
- CSS custom properties for theming
- Multiple color palettes (sand, tomato, blue, iris, jade)
- Light and dark mode support
- Consistent spacing, typography, and animations

## Usage

This is a standalone website that can be:
- Deployed to GitHub Pages
- Hosted on any static hosting service
- Served locally for development

Simply open `index.html` in a browser or serve it with any static file server.
